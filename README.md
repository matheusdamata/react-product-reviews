# REACTJS-REVIEWS 
Leitor de Reviews de Produtos com back-end local em MySql

<b>Banco de dados</b>: MySQL<br />
<b>Cliente</b>: DBeaver 22.2.0<br />

Rotas:
GET <b>/api/reviews</b> - Retorna todos os reviews inseridos na db.<br />
